# Rune — Learning Journal

A narrative record of what clicked, what confused, and what was interesting along the way.
Entries are newest-first within each phase. This is also the raw material for future blog posts.

**Entry tags:**
- `#aha` — a genuine insight that changed how you think about something
- `#mistake` — something that tripped you up (the technical detail lives in `findings.md`; the story lives here)
- `#concept` — an explanation of a Rust concept that finally made sense
- `#story` — an anecdote or narrative thread with potential blog value
- `#reference` — a pattern or fact worth being able to look up again

Each entry ends with:
- **One-sentence version:** — the insight compressed to a single line (for quick reference scanning)
- **Blog angle:** — how this story would be framed for an outside reader
